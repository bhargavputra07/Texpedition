# ⚡ Receptivity AI

A self-learning customer engagement engine that decides whether to send notifications based on user behavior, sentiment, engagement patterns, and adaptive machine learning.

## Features

* Rule-based receptivity scoring
* Online machine learning with SGDClassifier
* Adaptive per-customer thresholds
* Real-time feedback learning
* Explainable AI decisions
* Gradio interactive dashboard

## Architecture

User Signals → Rule Engine → ML Confidence → Hybrid Decision Engine → Notification Action

Feedback → Online Learning → Threshold Adaptation → Improved Future Decisions

## Tech Stack

* Python
* Scikit-Learn
* NumPy
* Pandas
* Gradio
